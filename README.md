## Descripción General

El presente portafolio reúne los proyectos desarrollados durante el semestre en la asignatura de Desarrollo de Aplicaciones Web. Su propósito es evidenciar los conocimientos, habilidades y competencias adquiridas mediante la creación de aplicaciones web funcionales, así como documentar las distintas etapas de análisis, diseño, implementación y prueba de cada proyecto realizado.

Cada proyecto incluido contiene información detallada sobre sus objetivos, problemática abordada, tecnologías empleadas, conceptos aplicados, funcionamiento, evidencias de ejecución y aprendizajes obtenidos. De esta manera, el portafolio sirve como un registro integral del trabajo realizado a lo largo del curso y como evidencia del crecimiento académico y técnico alcanzado.

Para el desarrollo de los proyectos se utilizaron tecnologías y herramientas fundamentales en el ámbito del desarrollo web, tales como PHP, HTML5, CSS3, JavaScript, MySQL, Visual Studio Code, GitHub, XAMPP y Moodle.

---

## Proyectos Incluidos

### Proyecto 1. Sistema de Registro de Eventos

Aplicación web desarrollada para registrar participantes de un evento mediante formularios web, validación de datos y almacenamiento de información en una base de datos MySQL.

### Proyecto 2. Mini Aplicación de Gestión de Usuarios, Contactos y Citas

Sistema web que permite registrar usuarios, administrar contactos y gestionar citas mediante módulos conectados a una base de datos relacional.

### Proyecto 3. Sistema de Gestión de Citas Médicas

Aplicación web con autenticación de usuarios, administración de citas médicas, historial de consultas y operaciones CRUD utilizando PHP y MySQL.
